# Synchro Games Wandering Trades

A simple mod to add new usefull trades to the wandering trader like armor trim templates and more attractive offers.
### Instalation

This mod is only compatible with fabric loaders

### Compatibility

Minecraft 1.21.4

### Requirements

Java 21
Fabric >=0.16.9
